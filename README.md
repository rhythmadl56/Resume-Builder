# 🚀 Resume Builder

A modern and responsive Resume Builder web application built using **React** and **Tailwind CSS**. This app allows users to create professional resumes with a live preview, choose from multiple templates, and download their resume as a PDF.

---

## 🔗 Live Demo

👉 **[View Live Project](https://makemyresume-byrhythm.vercel.app/)**

---

## ✨ Features

* 🧾 Multi-step form for structured resume input
* ⚡ Real-time resume preview while typing
* 🎨 Multiple templates (Simple & Modern)
* 📄 One-click PDF download
* 📱 Fully responsive design
* 💾 Local state management (no backend required)

---

## 🛠️ Tech Stack

* **Frontend:** React (Vite)
* **Styling:** Tailwind CSS
* **PDF Generation:** html2pdf.js
* **State Management:** React Hooks

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/rhythmadl56/Resume-Builder.git
cd Resume-Builder
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
